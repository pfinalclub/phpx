Run PHP Tools Like npx

phpx is a command-line tool written in Rust that allows you to run PHP command-line tools (typically .phar files) without requiring global or local persistent installation.

Get Started GitHub
Terminal

Installation Guide

Build from Source

# Clone the project
git clone https://github.com/pfinalcub/phpx.git
cd phpx

# Build the project
cargo build --release

# Install to system path
sudo cp target/release/phpx /usr/local/bin/

Download Pre-built Binaries

Download binaries for your platform from GitHub Releases:

Usage Examples

Basic Usage

# Run PHPStan for code analysis
phpx phpstan analyse src/

# Run PHP-CS-Fixer to format code
phpx php-cs-fixer fix file.php

# Use specific tool versions
phpx phpstan@^1.10 analyse --level=max src/

Cache Management

# Clean cache for a specific tool
phpx cache clean phpstan

# List cached tools
phpx cache list

# View cache details for a tool
phpx cache info phpstan

Advanced Options

# Force clear cache and re-download
phpx --clear-cache phpstan analyse src/

# Skip security verification
phpx --skip-verify php-cs-fixer fix file.php

# Specify PHP binary path
phpx --php /usr/local/bin/php8.1 phpstan
