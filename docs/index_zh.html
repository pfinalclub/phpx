像 npx 一样运行 PHP 工具

phpx 是一个用 Rust 编写的命令行工具，允许您直接运行 PHP 命令行工具（通常是 .phar 文件）， 而无需进行全局或本地的持久化安装。

开始使用 GitHub
终端
$ phpx phpstan analyse src/
正在分析代码...
$ phpx php-cs-fixer fix file.php
代码格式化完成！

核心特性

🚀

零污染运行

工具运行不污染全局或项目本地环境，所有依赖被隔离在专用缓存目录。

📦

智能缓存

自动下载并缓存工具，支持离线使用，提高后续执行速度。

🔄

版本管理

支持语义化版本约束和并行版本缓存，灵活管理工具版本。

🔒

安全验证

支持文件哈希验证，保障代码来源可信。

⚡

高性能

异步下载，利用缓存实现接近本地工具的启动速度。

🛠️

多源支持

支持 Packagist、GitHub Releases 和直接 URL 等多种源类型。

安装指南

从源码构建

# 克隆项目
git clone https://github.com/pfinalclub/phpx.git
cd phpx

# 构建项目
cargo build --release

# 安装到系统路径
sudo cp target/release/phpx /usr/local/bin/

下载预编译二进制

从 GitHub Releases 下载对应平台的二进制文件：

Linux x86_64 macOS x86_64 macOS ARM64 Windows

使用示例

基本使用

# 运行 PHPStan 进行代码分析
phpx phpstan analyse src/

# 运行 PHP-CS-Fixer 格式化代码
phpx php-cs-fixer fix file.php

# 使用特定版本的工具
phpx phpstan@^1.10 analyse --level=max src/

缓存管理

# 清理指定工具的缓存
phpx cache clean phpstan

# 查看已缓存的工具
phpx cache list

# 查看工具缓存详情
phpx cache info phpstan

高级选项

# 强制清除缓存并重新下载
phpx --clear-cache phpstan analyse src/

# 跳过安全验证
phpx --skip-verify php-cs-fixer fix file.php

# 指定 PHP 二进制路径
phpx --php /usr/local/bin/php8.1 phpstan
